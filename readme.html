What it does

This add-on displays text within your admin page's header to let the signed-on admin know when his/her password was last changed, giving them some amount of implicit warning as to the need to change their password. The display (as of v1.1.0) also includes the admin's previous login date and IP address from which the admin logged in.

Installation

There are four core-file overwrites in this mod; you should always backup your cart’s database and files prior to making any changes.

Copy the admin files to your cart, after renaming the "YOUR_ADMIN" directory to match your custom admin’s name:

  1. /YOUR_ADMIN/includes/header.php
  2. /YOUR_ADMIN/includes/auto_loaders/config.zc154_compatibility.php
  3. /YOUR_ADMIN/includes/functions/admin_access.php
  4. /YOUR_ADMIN/includes/init_includes/init_zc154_compatibility.php
  5. /YOUR_ADMIN/includes/languages/english.php

Note: If you have made previous modifications to these core-files, you will need to use WinMerge (or an equivalent program) to merge the changes. All changes for this add-on are tagged with //-bof-lat9 and //-eof-lat9.
